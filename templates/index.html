Reflection and Learnings

Throughout my journey in this course (CPE 232: Managing Enterprise Servers), I was able to create a workflow using infrastructure as code practices to build an enterprise environment in Oracle VM VirtualBox using Debian and CentOS distributions. I was able to orchestrate remote servers using SSH, which allows me to install and communicate securely and remotely. I also learned about the DevOps culture by utilizing GitHub and leveraging infrastructure as code with Ansible. As the course progressed, I evaluated real-world enterprise scenarios virtually, finding solutions to enterprise problems and determining the best services to use in the industry. Additionally, I successfully implemented OpenStack on my virtual machine, which offers various cloud services. Lastly, I would like to thank my instructor, Dr. Jonathan V. Taylar, for his continuous support, expert guidance in managing enterprise servers, and creating a positive and low-pressure class environment.
